Github Pages

If you need to host a static website, (like this), there's no better choice to use github pages.
With tools like WuFoo or Typeform you can allow people to contact you.


Having a static website is no issue, if you know how to do it.


Go to Github

App:
Heroku

Heroku is the most used cloud platform around.
You can start small, spending 0$ hosting your app.

Base limits are low, but enough for starting.


Go to Heroku

App:
Appfog

Appfog is an alternative to heroku.
It has higher limits, allowing to stay in the free version for more.

Pushing code is less fast then Heroku (still deceptively simple).


Go to Appfog
